# imc-calculator
Calcula el IMC usando la fórmula:

IMC = peso / altura²

Clasifica el resultado en:
Bajo peso
Normal
Sobrepeso
Obesidad
Muestra el resultado de forma clara para el usuario

Tecnologías usadas
Python

Cómo usarlo
Ejecuta el archivo .py
Ingresa:
Tu peso (kg)
Tu altura (m)
Obtendrás:
Tu IMC
Tu clasificación

Ejemplo
Entrada:
Peso: 70 kg
Altura: 1.75 m

Salida:
IMC: 22.86
Clasificación: Normal
